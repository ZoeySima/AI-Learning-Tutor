# AI Learning Tutor — 用 AI 重新定义学习

一个基于 Claude 的个性化学习助手，帮你快速建立对陌生领域的「概念性理解」。

## ✨ What's New in v0.2.0

**v0.2.0 带来了基于学习科学的 6 项重大改进**：

### 🎯 智能验证，杜绝"假懂"
- **推理路径验证**：不只看答案对错，还要验证你的思考过程
- **变式练习**：答错时自动生成不同情境的题目，验证真实理解
- **尝试次数追踪**：记录每道题的答题历史

### 🔍 事实核查，确保准确性
- **关键领域检测**：金融、医学、法律等敏感话题自动启用双模型验证
- **自动纠错**：发现事实错误立即重新生成内容
- **免责声明**：敏感内容自动添加适当警告

### 📋 诊断性前测，真正个性化
- **先备知识评估**：学习前先做 3-5 道诊断题
- **动态调整深度**：根据前测结果自动调整章节难度
- **精准定位**：AI 分析你的知识盲区并推荐重点

### 📊 结构化笔记，形成学习资产
- **知识图谱**：Mermaid 图表展示概念关系
- **可折叠章节**：完整内容默认折叠，按需展开
- **掌握度追踪**：可视化进度条 + 薄弱点分析
- **对比表格**：三视角内容并排对比

### 🔄 间隔重复，对抗遗忘
- **螺旋回顾**：每章开始前自动回顾前 1 章和前 3 章
- **闪卡生成**：AI 自动生成快速回顾题
- **科学记忆**：基于艾宾浩斯遗忘曲线的复习策略

### 🎨 可视化学习，降低认知负荷
- **ASCII 图表**：流程图、树状结构、对比表
- **进度可视化**：章节树 + 完成状态
- **视觉层次**：图标、颜色、格式化增强可读性

## 核心方法论

本工具实现了一套经过验证的 AI 学习四步法：

1. **Map**：生成全局认知地图（章节大纲），先建立框架
2. **Learn**：逐章深挖，每章用「三视角 + 真实例子」讲解
3. **Quiz**：每章末出题验证，答对才进下一章
4. **Loop**：循环直到所有章节完成

## 为什么有效？

- **个性化定制**：诊断前测 + 根据背景调整讲解颗粒度
- **主动验证**：推理路径验证 + 变式练习，暴露理解盲区
- **结构化记忆**：先建地图再填细节 + 间隔重复巩固
- **信任基石**：关键领域事实核查，确保学习内容准确

## 快速开始

### 安装

```bash
# 克隆仓库
git clone https://github.com/ZoeySima/AI-Learning-Tutor.git
cd ai-learning-tutor

# 安装依赖
pip install -r requirements.txt

# 或者使用 pip 安装（开发模式）
pip install -e .
```

### 配置 API Key

```bash
export ANTHROPIC_API_KEY="sk-ant-..."
```

### 开始学习

```bash
# 开始新学习
ai-tutor start "区块链技术基础"

# 恢复已有会话
ai-tutor resume 20260527-blockchain

# 列出所有会话
ai-tutor list

# 导出学习笔记
ai-tutor export 20260527-blockchain -o notes.md
```
